# Chrodis 快捷键

## 传输

| 快捷键 | 动作 |
| --- | --- |
| `Space` | 播放/暂停 |
| `Enter` | 停止 |
| `Cmd+R` | 开始/停止录音 |

## 编辑

| 快捷键 | 动作 |
| --- | --- |
| `Cmd+Z` | 撤销 |
| `Shift+Cmd+Z` | 重做 |
| `Cmd+C` / `Cmd+V` | 复制/粘贴 |
| `Cmd+D` | 复制一份 |
| `Delete` | 删除当前焦点中的音符、片段或轨道 |

## 选择

| 快捷键 | 动作 |
| --- | --- |
| `Cmd+A` | 全选当前区域：时间线全选片段，钢琴卷帘全选音符 |
| `Shift+点击` | 追加选择 |
| `Cmd+点击` | 切换选择 |

## 工具

| 快捷键 | 动作 |
| --- | --- |
| `1` | 指针工具 |
| `2` | 框选工具 |
| `3` | 剪刀工具 |

## 缩放

| 快捷键 | 动作 |
| --- | --- |
| `Cmd++` / `Cmd+-` | 水平放大/缩小 |
| `Option+双指上下滚动` | 垂直放大/缩小轨道或音符行高 |
| `Cmd+0` | 重置缩放 |

## 窗口

| 快捷键 | 动作 |
| --- | --- |
| `Cmd+,` | 打开偏好设置 |
| `Esc` | 关闭菜单、框选或编辑器浮层 |
